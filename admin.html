MY PROJECTS

SELECTIVE TEXT EXTRACTION

One of the most untouched topics in the field of Image Analysis is that of automated detection of underlines in a document image. The conventional problem involves an image of a printed document (usually taken from a flat-bed scanner) with paragraph texts that are manually underlined by a pen/pencil. Previous attempts have only been aimed at detection of the underlines. The techniques mentioned in literature involved use of Hough Transform, use of edge detection and bounding box estimation algorithms. However, most of these techniques had a major problem with accuracy and reliability since they generated false positive indications of an underline and were not able to take into account the local curvatures and bends that a manual underline may have. To overcome these issues, we began with a novel and heuristic approach towards this problem. The document scan was initially pre-processed. It involved thresholding the image using Otsu’s method which resulted into a binary image. It was followed by convolving it with a filter designed to enhance the horizontal edges in the image. To identify those horizontal edges which may have corresponded to an underline, we assigned weights to each pixel of that image. These weights were distributed in the following way – A classical 5*5 window sliding technique was used where the window was slid from that pixel onward in the horizontal direction. Based on the number of pixels that followed a straight line, a weight was assigned to the original pixel. An appropriate threshold was then used to ignore those pixels with very less weight as those were undesirable. After that, we used the location of endpoints to extract the underlined texts from the original image. Read More...

VIRTUAL MEMORY MANAGEMENT

Our ojective was to simulate and analyze the performance of page replacement techniques: LRU, clock, and FIFO by varying the page size.The following algorithms were implemented.
FIFO, LRU, Clock
The following features were implemented
1. Variable page size 2. Local & Global Page Replacement Strategies 3. Prepaging

LRU algorithm for page replacement gives the minimum number of page faults in comparison to the other algorithms such as FIFO and Clock. FIFO and Clock algorithms thus go by Belady's anomaly, thereby showing a peak in the middle of decreasing curve. Global page replacement policy seems to work more efficiently as compared to local page replacement policy, as it is free to select any page from memory. On the whole, algorithms with global page replacement policy and prepaging enabled show the minimum number of page faults as compared to other possible combinations of both with LRU giving the least number of page faults.
